# 日本国憲法

昭和21年11月3日公布／昭和22年5月3日施行

## 構造

日本国憲法を純自然言語で構造化したリポジトリ。

形式 | 値
-----|-----
言語 | 日本語（旧字体）のみ
書式 | Markdown
階層 | 章 = ディレクトリ、条 = ファイル
条数 | 前文 + 全103条（11章）
連番 | 各ファイルは `nn-nnn.md` 形式（通番-条数）

## スタイル

**純正樹形構造** — 章をディレクトリ、条をファイルに割り当てることで、
法体系の階層をそのままファイルシステムに写像している。

- `# 第N条` をファイル内唯一の見出しとする
- 項（②③...）は空行で区切る
- 一号列挙（一 二 三...）はそのまま記述
- メタデータ・YAML・コードを一切含まない純自然言語
